How to Edit (Homepage) Slider

You can edit, add or delete slider slides with great ease using "Templateism Admin Panel"

    1. Go to Blogger >> Layout >> Slider >> Add a Gadget
    2. From the list Select "Image"
    3. Upload Slider Image, Add Title, Caption and Link if needed
    4. Save the Widget, Press "Save" Button

How to Edit (Homepage) Featured Heading

To edit the featured heading on the homepage, follow the following instructions:

    1. Go to Blogger >> Template >> Edit HTML >> Search For <!--Latest Posts-->
    2. Customize the heading according to your needs
    3. Save the template, press "Save template" button

How to Edit Feature Category with Thumbnail (Homepage)

If you want to display the 2 posts with thumbnails that appear in the homepage of theme demo, do as follows:

    1. Go to Blogger >> Layout >> Featured Posts >> EDIT
    2. Now, in the text area write the name of the label. It will display the posts from only that label which you'll write.
    3. Save the template, press "Save template" button
